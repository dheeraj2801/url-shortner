## Steps to run the server and client


1. Go to the backend folder

```
cd backend
```

2. Install dependencies

```
npm install
```

3. Run the express server using node

```
node index.js
```


4. Go to the client

```
cd client/frontend
```

6. Install dependencies

```
npm install
```

7. Run the react server

```
npm run dev
```
